Github Repository

Functionality

To use the tool, there are only a few simple commands

  • task-cli add {DESCRIPTION}

  • task-cli update {ID} {DESCRIPTION}

  • task-cli mark-in-progress {ID}

  • task-cli mark-done {ID}

  • task-cli delete {ID}
